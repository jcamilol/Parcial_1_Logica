    \begin{itemize}
        \item $\forall x \neg(x < 0)$
        \item $\forall x \forall y \forall z ((x < y \wedge y < z) \rightarrow x < z)$
        \item $\forall x \exists y (x < y)$
    \end{itemize}
    Sea $c$ un nuevo símbolo de constante. Muestre que $T_c = T \cup \{ \neg(c < 0) \}$ es consistente. \\
